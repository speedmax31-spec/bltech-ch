==============================================
   나의 개인 PMS 사용법
==============================================

[ 실행하기 ]
  1. 이 폴더(test1)의  start.bat  파일을 더블클릭하세요.
  2. 검은 창이 뜨고, 잠시 후 브라우저가 자동으로 열립니다.
  3. 끝! 이제 자유롭게 할 일을 추가/수정/완료하면 됩니다.

[ 종료하기 ]
  - 검은 창(콘솔)을 닫으면 종료됩니다.
  - 브라우저 탭만 닫아도 데이터는 이미 저장돼 있으니 안전합니다.

[ 데이터는 어디에 저장되나요? ]
  - 이 폴더의  data.json  파일에 실제로 저장됩니다.
  - 매일 자동 백업이  backups  폴더에 보관됩니다(최근 14일).
  - 다른 컴퓨터로 옮기려면 이 폴더 전체를 복사하면 됩니다.

[ 자주 쓰는 기능 ]
  - 추가   : 각 칸 아래 입력 후 Enter
  - 완료   : 왼쪽 체크박스 클릭
  - 수정   : 글자를 더블클릭 → 고치고 Enter (취소는 Esc)
  - 중요   : ★ 버튼
  - 삭제   : 휴지통 버튼
  - 일정   : '개인 일정'은 날짜를 같이 넣을 수 있어요

[ 화면 위쪽 저장 상태 표시 ]
  - "파일에 저장됨"  : 정상 (data.json 에 저장 중)
  - "이 브라우저에만 저장" : start.bat 없이 파일을 직접 열었을 때
                            → 가급적 start.bat 으로 여세요.

[ 바탕화면에 바로가기 만들기(선택) ]
  - start.bat 을 마우스 오른쪽 클릭 → 보내기 → 바탕화면(바로 가기 만들기)
